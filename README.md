# Machine Learning Projects Repository

This repository contains a collection of machine learning work, organized into folders by project type — ranging from algorithms implemented from scratch, to applied assignments on real-world datasets, to a focused model comparison on a medical dataset.

## Repository Structure

```
repository/
├── ml-from-scratch/          # ML algorithms built from scratch
├── assignments/           # Assignment-based projects on various datasets
├── heart-disease-prediction/ # Model comparison on heart.csv dataset
└── README.md                
```

## Projects Overview

### 1. [ML Algorithms from Scratch](./algo-from-scratch)
Implementations of core ML algorithms built using only NumPy — no ML libraries — to understand the underlying math and logic.

- **Algorithms:** KNN, Linear Regression, Logistic Regression
- **Focus:** Manual implementation of gradient descent, distance metrics, and cost functions

### 2. [ML Assignments](./assignments)
Applied ML projects using standard libraries (scikit-learn, pandas) across different real-world datasets.

- **Projects:** E-Commerce, House Price Prediction, Iris Flower Classification, Employee Turnover Prediction
- **Focus:** End-to-end ML workflow — EDA, preprocessing, model building, evaluation

### 3. [Heart Disease Prediction](./implement)
Comparison of multiple classification models on the `heart.csv` patient dataset.

- **Algorithms:** KNN, Logistic Regression, Naive Bayes
- **Focus:** Model comparison and evaluation on medical/health data

## Tech Stack

| Tool | Purpose |
|---|---|
| Python | Core programming language |
| NumPy | Numerical computation (used heavily in from-scratch models) |
| Pandas | Data manipulation |
| scikit-learn | ML models and evaluation metrics |
| Matplotlib / Seaborn | Data visualization |
